# 储值方案集成技术方案

## 1. 方案概述

### 1.1 目标

将新设计的储值方案（CrmDepositPlan）集成到现有的储值流程中，替代旧的储值规则（CrmSaveRule）。

### 1.2 核心改动

1. **储值提交接口改造**：支持储值方案和档位选择
2. **储值成功回调改造**：支持多券、券包、积分、会员升级
3. **购买限制检查**：实施单人周期限制和总量限制
4. **统计数据更新**：更新方案的销售次数和金额

### 1.3 兼容性策略

**双轨运行：**
- 保留旧的 CrmSaveRule 流程（向后兼容）
- 新增 CrmDepositPlan 流程
- 通过参数区分使用哪种方案

## 2. 数据流程设计

### 2.1 当前流程（CrmSaveRule）

```
用户选择储值规则
    ↓
ChargeService.commit()
    ├─ ruleType = 0（充值套餐）
    ├─ ruleType = -1（小额储值）
    └─ ruleType = -2（霸王餐储值）
    ↓
查询 CrmSaveRule
    ↓
计算金额
    ├─ principalAmount（本金）
    ├─ giveAmount（赠送金额）
    ├─ points（赠送积分）
    └─ giveCouponId（赠送券ID）
    ↓
创建支付订单
    ↓
创建 CrmDealTask（储值任务）
    ↓
用户支付
    ↓
支付回调 → CardBalanceService
    ↓
更新余额 + 发放积分 + 发放优惠券 + 升级会员
```

### 2.2 新流程（CrmDepositPlan）

```
用户选择储值方案
    ↓
用户选择储值档位
    ↓
ChargeService.commitWithPlan()（新接口）
    ├─ planLid（方案ID）
    └─ tierIndex（档位索引）
    ↓
查询 CrmDepositPlan
    ↓
验证方案可用性
    ├─ 有效期检查
    ├─ 周期规则检查
    ├─ 时段规则检查
    ├─ 门店规则检查
    ├─ 会员等级检查
    └─ 购买限制检查
    ↓
获取档位配置
    ├─ saveAmount（储值金额）
    ├─ giftAmount（赠送金额）
    ├─ giftPoints（赠送积分）
    ├─ coupons[]（赠送券列表）
    ├─ couponPackages[]（赠送券包列表）
    └─ isUpgradeMemberLevel（是否升级）
    ↓
创建支付订单
    ↓
创建 CrmDealTask（储值任务）
    ├─ 保存方案ID
    ├─ 保存档位索引
    └─ 保存赠送内容（JSON）
    ↓
用户支付
    ↓
支付回调 → CardBalanceService
    ↓
更新余额 + 发放积分 + 批量发放券 + 批量发放券包 + 升级会员
    ↓
更新方案统计数据
```

## 4. 接口设计

### 4.1 储值提交接口（新）

**接口：** `POST /api/crm/charge/commitWithPlan`

**请求参数：**
```java
public class ChargeCommitWithPlanRequest {
    private Long mid;              // 商户ID
    private Long sid;              // 门店ID
    private Long cardLid;          // 会员卡ID
    private Long planLid;          // 储值方案ID
    private Integer tierIndex;     // 档位索引（从0开始）
    private String paymentMethod;  // 支付方式（H5/MINI_PROGRAM）
    private String channel;        // 渠道代码
    private String returnUrl;      // 支付回调地址
}
```

**响应参数：**
```java
public class ChargeCommitResponse {
    private String tradeNo;        // 交易流水号
    private String paymentUrl;     // 支付URL
    private BigDecimal amount;     // 支付金额
    private Long taskLid;          // 储值任务ID
}
```

**业务逻辑：**
```java
@Transactional
public ChargeCommitResponse commitWithPlan(ChargeCommitWithPlanRequest request) {
    // 1. 验证会员卡
    CrmCard card = validateCard(request.getCardLid());

    // 2. 查询储值方案
    CrmDepositPlan plan = depositPlanService.getById(request.getPlanLid());
    if (plan == null || plan.getStatus() != 1) {
        throw new BizException("储值方案不存在或已禁用");
    }

    // 3. 验证方案可用性
    validatePlanAvailability(plan, request);

    // 4. 获取档位配置
    DepositTierRule tierRule = plan.getTierRule();
    if (tierRule == null || tierRule.getTiers() == null) {
        throw new BizException("储值方案未配置档位");
    }
    if (request.getTierIndex() < 0 || request.getTierIndex() >= tierRule.getTiers().size()) {
        throw new BizException("档位索引无效");
    }
    DepositTier tier = tierRule.getTiers().get(request.getTierIndex());

    // 5. 检查首充限制
    if (tier.getIsFirstChargeOnly() && hasCharged(card.getLid())) {
        throw new BizException("该档位仅限首充");
    }

    // 6. 检查购买限制
    validatePurchaseLimit(plan, card);

    // 7. 计算金额
    BigDecimal saveAmount = tier.getSaveAmount();
    BigDecimal giftAmount = tier.getGiftAmount() != null ? tier.getGiftAmount() : BigDecimal.ZERO;
    BigDecimal totalAmount = saveAmount; // 支付金额 = 储值金额（不包含赠送）

    // 8. 创建支付订单
    String tradeNo = createPaymentOrder(totalAmount, request.getPaymentMethod());

    // 9. 创建储值任务
    CrmDealTask task = new CrmDealTask();
    task.setLid(IdWorkerPlus.getId());
    task.setMid(request.getMid());
    task.setSid(request.getSid());
    task.setCardLid(card.getLid());
    task.setDepositPlanLid(plan.getLid());           // ⭐ 新字段
    task.setDepositTierIndex(request.getTierIndex()); // ⭐ 新字段
    task.setDepositTierSnapshot(JSON.toJSONString(tier)); // ⭐ 新字段（快照）
    task.setPrincipalAmount(saveAmount);
    task.setGiveAmount(giftAmount);
    task.setGivePoint(tier.getGiftPoints());
    task.setAmount(totalAmount);
    task.setTradeNo(tradeNo);
    task.setTradeState(TradeStateEnum.USERPAYING);
    task.setCreatedTime(LocalDateTime.now());
    dealTaskService.save(task);

    // 10. 发送延时关单消息（10分钟）
    sendCloseOrderMessage(task.getLid(), 10 * 60 * 1000);

    // 11. 返回支付信息
    return new ChargeCommitResponse()
        .setTradeNo(tradeNo)
        .setPaymentUrl(getPaymentUrl(tradeNo))
        .setAmount(totalAmount)
        .setTaskLid(task.getLid());
}
```

### 4.2 储值成功回调改造

**方法：** `CardBalanceService.processDepositPlan()`

**业务逻辑：**
```java
@Transactional
public void processDepositPlan(CrmDealTask task) {
    // 1. 验证任务状态
    if (!TradeStateEnum.USERPAYING.equals(task.getTradeState())) {
        log.warn("任务状态不是待支付，跳过处理");
        return;
    }

    // 2. 获取档位快照
    String snapshotJson = task.getDepositTierSnapshot();
    DepositTier tier = JSON.parseObject(snapshotJson, DepositTier.class);

    // 3. 更新会员余额
    updateCardBalance(task, tier);

    // 4. 发放积分
    if (tier.getGiftPoints() != null && tier.getGiftPoints() > 0) {
        givePoints(task.getCardLid(), tier.getGiftPoints(), task.getLid());
    }

    // 5. 批量发放优惠券
    if (tier.getCoupons() != null && !tier.getCoupons().isEmpty()) {
        giveCoupons(task, tier.getCoupons());
    }

    // 6. 批量发放券包
    if (tier.getCouponPackages() != null && !tier.getCouponPackages().isEmpty()) {
        giveCouponPackages(task, tier.getCouponPackages());
    }

    // 7. 升级会员等级
    if (Boolean.TRUE.equals(tier.getIsUpgradeMemberLevel())
        && tier.getUpgradeMemberLevelLid() != null) {
        upgradeMemberLevel(task.getCardLid(), tier.getUpgradeMemberLevelLid());
    }

    // 8. 更新任务状态
    task.setTradeState(TradeStateEnum.SUCCESS);
    task.setSuccessTime(LocalDateTime.now());
    dealTaskService.updateById(task);

    // 9. 更新方案统计数据
    updatePlanStatistics(task.getDepositPlanLid(), tier.getSaveAmount());

    // 10. 发送通知
    sendSuccessNotification(task);
}

// 批量发放优惠券
private void giveCoupons(CrmDealTask task, List<CouponItem> coupons) {
    List<CouponIssuanceDTO> couponList = new ArrayList<>();

    for (CouponItem couponItem : coupons) {
        CouponIssuanceDTO dto = new CouponIssuanceDTO();
        dto.setCouponCode(couponItem.getCouponLid());
        dto.setCouponNum(couponItem.getIssueQtyPerTime());
        couponList.add(dto);
    }

    if (!couponList.isEmpty()) {
        CouponOpAddDTO couponOpAddDTO = new CouponOpAddDTO();
        couponOpAddDTO.setMid(task.getMid());
        couponOpAddDTO.setLid(task.getCardLid());
        couponOpAddDTO.setCouponList(couponList);
        couponOpAddDTO.setCouponChannel(CouponChannelEnum.Recharge);
        couponOpAddDTO.setOrderId(String.valueOf(task.getLid()));

        crmCouponOpServicePlus.couponOpAdd(couponOpAddDTO);
    }
}

// 批量发放券包
private void giveCouponPackages(CrmDealTask task, List<CouponPackageItem> packages) {
    List<CouponIssuanceDTO> packageList = new ArrayList<>();

    for (CouponPackageItem packageItem : packages) {
        CouponIssuanceDTO dto = new CouponIssuanceDTO();
        dto.setCouponCode(packageItem.getPackageLid());
        dto.setCouponNum(packageItem.getPackageQty());
        packageList.add(dto);
    }

    if (!packageList.isEmpty()) {
        CouponOpAddDTO couponOpAddDTO = new CouponOpAddDTO();
        couponOpAddDTO.setMid(task.getMid());
        couponOpAddDTO.setLid(task.getCardLid());
        couponOpAddDTO.setCouponList(packageList);
        couponOpAddDTO.setCouponChannel(CouponChannelEnum.Recharge);
        couponOpAddDTO.setOrderId(String.valueOf(task.getLid()));

        crmCouponOpServicePlus.couponOpAdd(couponOpAddDTO);
    }
}

// 更新方案统计数据
private void updatePlanStatistics(Long planLid, BigDecimal amount) {
    depositPlanService.update(
        new LambdaUpdateWrapper<CrmDepositPlan>()
            .setSql("total_sold_count = total_sold_count + 1")
            .setSql("total_sold_amount = total_sold_amount + " + amount)
            .eq(CrmDepositPlan::getLid, planLid)
    );
}
```

## 5. 购买限制实现

### 5.1 单人周期限制检查

```java
private void validatePurchaseLimit(CrmDepositPlan plan, CrmCard card) {
    PurchaseLimitRule limitRule = plan.getPurchaseLimitRule();
    if (limitRule == null || !Boolean.TRUE.equals(limitRule.getEnabled())) {
        return;
    }

    // 单人周期限制
    if ("PERSON_CYCLE".equals(limitRule.getLimitType())) {
        LocalDateTime startTime = getStartTimeOfCycle(limitRule.getCycleType());
        LocalDateTime endTime = LocalDateTime.now();

        long count = dealTaskService.count(
            new LambdaQueryWrapper<CrmDealTask>()
                .eq(CrmDealTask::getCardLid, card.getLid())
                .eq(CrmDealTask::getDepositPlanLid, plan.getLid())
                .eq(CrmDealTask::getTradeState, TradeStateEnum.SUCCESS)
                .between(CrmDealTask::getSuccessTime, startTime, endTime)
        );

        if (count >= limitRule.getLimitValue()) {
            throw new BizException(
                String.format("该方案在当前周期内最多购买%d次", limitRule.getLimitValue())
            );
        }
    }

    // 总量限制
    if (limitRule.getTotalLimit() != null && limitRule.getTotalLimit() > 0) {
        if (plan.getTotalSoldCount() >= limitRule.getTotalLimit()) {
            throw new BizException("该方案已售罄");
        }
    }
}

private LocalDateTime getStartTimeOfCycle(String cycleType) {
    LocalDateTime now = LocalDateTime.now();
    switch (cycleType) {
        case "DAY":
            return now.with(LocalTime.MIN);
        case "WEEK":
            return now.with(DayOfWeek.MONDAY).with(LocalTime.MIN);
        case "MONTH":
            return now.withDayOfMonth(1).with(LocalTime.MIN);
        default:
            throw new BizException("不支持的周期类型");
    }
}
```
