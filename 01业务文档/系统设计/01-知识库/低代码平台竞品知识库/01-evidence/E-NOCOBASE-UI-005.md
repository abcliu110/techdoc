# 证据：NocoBase Custom Action Event 与页面按钮绑定

## 原始观察

官方 Custom Action Event 文档说明，当内置 Add New、Delete、Edit、View 等数据动作不能满足复杂业务时，可以在 workflow 中使用 Custom Action Event，并将其绑定到页面 block 中的 Trigger Workflow 按钮。用户点击按钮后触发自定义动作工作流。

## 可抽取知识

- NocoBase 把页面按钮与 workflow 触发器打通。
- 自研平台需要把“动作按钮”建模为可触发流程的命令，而不是普通 UI 元素。
