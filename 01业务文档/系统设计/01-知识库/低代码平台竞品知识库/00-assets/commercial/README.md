# 商业软件截图证据目录

本目录保存商业低代码平台和企业 UI 组件的公开界面截图证据。

## 收录规则

- 优先保存官方产品、官方文档、官方演示或官方视频中可辨认的软件界面。
- 不把营销封面、品牌插画或无产品界面的页面作为设计器视觉证据。
- 每张截图必须在对应证据卡中记录产品、来源 URL、页面标题、捕获日期和可见内容。
- 截图只证明画面中可见的结构与状态，不自动证明隐藏能力或内部实现。
- 设计推导必须与截图事实分开。
