## 横向结论

1. 金蝶 AI 苍穹在企业业务元模型、组织权限、流程、动态表单和 ERP 语义上最强；本轮公开资料深挖增强了 KDDM、动态表单、单据设计、在线表单设计平台、流程服务和集成服务证据，但内部实现仍只能标高可信推断。
2. Frappe 是开源体系里最接近企业业务低代码的路线，但 UI Builder 体验不如内部工具平台。
3. Appsmith、ToolJet、Lowcoder 在 UI + 数据源 + 查询绑定上强，适合内部工具；Appsmith 本轮新增 Widget Development API、动态绑定和导出权限边界证据。
4. NocoBase 在数据模型、插件、工作流和业务应用之间平衡较好。
5. NocoDB 和 Directus 更适合作为数据平台/后台/API/权限参考；Directus 本轮新增 Fields、Access Control、ItemsService、Role/Policy/Access、Flows 证据，权限与服务层参考价值上升。
6. 表单设计器专题新增 Jotform、Typeform、Airtable、SurveyJS、Formstack、Form.io 作为横向参照：Jotform/SurveyJS/Form.io 适合学习规则与 schema 化，Typeform 适合学习分支路径体验，Airtable 提醒“条件隐藏不是权限”，Formstack 适合学习表单提交后的审批状态链。
