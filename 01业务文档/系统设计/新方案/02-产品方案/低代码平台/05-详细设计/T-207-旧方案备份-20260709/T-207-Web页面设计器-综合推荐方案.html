核心能力模块

五大能力模块，对应评审中最强的五个方案

G Web 布局引擎

基于 CSS Grid/Flex 的可视化布局设计，覆盖所有 Web 布局模式。Grid Area 可视化辅助线，吸附到最近网格交点。

  • GridLayout — 24 列栅格、area 命名布局
  • FlexLayout — wrap / grow / shrink / basis
  • DockLayout — top/left/right/bottom/fill 停靠
  • SplitPane — 可拖拽手柄、比例约束
  • StickyRegion — 固定表头/底栏
  • Portal Overlay — 统一浮层层级管理

T 母子嵌套表格

企业分录表格的核心能力。支持冻结列、嵌套批次/序列号子表、虚拟滚动、键盘导航、父子汇总。

  • EntryTable — 分录表格主组件
  • BatchDetail — 批次明细嵌套行
  • SerialDetail — 序列号子表
  • 列配置 — freeze/edit/sum/format
  • 虚拟滚动 — 10 万行级性能保障
  • 主子汇总 — 自动跨行聚合计算

Q 双视图质量管控

设计态和运行态同屏对照，发布前质量门禁检查：性能指标、权限裁剪、规则耗时、布局校验。

  • 设计态 + 运行态并排预览
  • 渲染耗时 / API 调用数监控
  • 规则执行耗时分析
  • 权限裁剪模拟（按角色）
  • 布局断点校验（1280/1440/1600/1920）
  • 发布门禁 — blocker 必须清零

D 字段驱动绑定

业务对象字段拖入页面，自动生成控件、列定义、校验规则、权限绑定和数据血缘关系。

  • BO 字段树 — 层级化业务对象
  • ReferencePicker — 基础资料选择
  • MoneyInput — 金额字段（含币种）
  • 自动校验 — required / range / format
  • 权限绑定 — 字段级 read/write
  • 数据血缘 — 影响分析可视化

R PC 响应式

专注企业 PC 多分辨率：1280/1440/1600/1920。解决字段换行、表格横滚、侧栏折叠等真实问题。

  • 4 档断点独立配置
  • aside collapse 折叠动画
  • 表格宽度自适应 + 横向滚动
  • 字段 span 跨列动态调整
  • 断点预览即时生效
  • 断点切换不丢失编辑状态

W 规则与审批联动

选中字段即可看到联动规则、审批路线、权限影响和发布风险。字段变更触发的规则链可视化。

  • onChange 事件 → 字段联动
  • beforeSubmit → 审批路线
  • 字段变更 → 价格/信用影响分析
  • 规则依赖拓扑图
  • 发布影响预估
  • 规则冲突检测

交互设计规范

参考 DevExpress + 金蝶 + VS 三大平台最佳实践
交互场景 行为规范 参考来源
拖放组件 从左侧工具箱拖出时，画布显示蓝色虚线投放区（drop zone），吸附到最近网格交点后显示 snap line 辅助线，松手后组件自动插入对应位置 DevExpress LayoutControl 对齐 + VS Designer 投放
选中组件 单击选中显示蓝色边框 + box-shadow，4 个角显示调整手柄（可拖拽改变 span），右侧属性面板自动切换到该组件的属性 VS WinForms Designer 选中态 + DevExpress Selection
调整尺寸 拖拽调整手柄时，Grid 容器内组件的 span-X 自动更新；Flex 容器内组件的 flex-grow/flex-basis 实时变化；右侧属性面板同步显示新值 VS TableLayoutPanel + DevExpress 自适应布局
大纲视图 左侧大纲（Outline）Tab 显示组件树层级，点击大纲项自动选中画布对应组件并滚动到可见区域，支持拖拽调整嵌套关系 VS Document Outline + DevExpress 组件树
属性编辑 右侧属性面板支持直接编辑数值，编辑后画布实时更新（无需确认）；部分属性提供下拉选择（如 display: grid/flex） VS PropertyGrid + DevExpress OptionsForm
断点预览 点击工具栏"断点"按钮，画布从 1440 切换到目标分辨率，布局重新计算，属性面板显示该断点下的差异项；不影响编辑状态的原始值 浏览器 DevTools Responsive + 金蝶多分辨率适配
双视图切换 点击"双视图"按钮，中央面板水平分栏：左为设计态（可编辑），右为运行态（只读预览 + 监控指标面板）；刷新运行态保持当前设计态位置 DevExpress 预览模式 + 自研性能监控
校验反馈 底部错误列表实时更新；Error 行红色高亮，点击定位到对应组件；发布按钮在存在 blocker 时禁用，悬停显示阻断原因 VS Error List + 金蝶单据发布检查
Undo/Redo 支持 Ctrl+Z / Ctrl+Y，多步撤销；工具栏撤销/重做按钮显示步数角标；Schema 变更前自动保存 checkpoint VS + DevExpress Command History
键盘导航 方向键切换选中相邻组件；Delete 删除选中组件；Alt+拖动 复制组件；Ctrl+S 保存；F5 运行预览 VS Designer 快捷键 + DevExpress Keyboard

模式切换机制

四种设计模式，适应不同用户和使用阶段

🎯 默认模式（01 企业单据）

DevExpress 经典三区结构。左侧工具箱，中央画布，右侧属性。适合业务实施人员快速上手。

新手指引 快速配置 单据页面

📐 布局模式（04 强布局）

左侧切换到"容器"Tab，右侧属性切换到"布局"Tab。画布显示 Grid Area 辅助线。适合复杂 Web 应用页面。

强布局 CSS Grid/Flex Dock/Split

📊 表格模式（05 母子嵌套）

中央画布切换到 EntryTable 设计视图。左侧切换到"表格树"Tab。右侧切换到"列属性"Tab。支持批次/序列号嵌套配置。

强表格 母子嵌套 虚拟滚动

👁 双视图模式（10 质量管控）

中央画布水平分栏：左设计右运行。右侧监控面板展示渲染耗时、API 调用、权限裁剪、规则耗时。适合发布前质量检查。

质量门禁 性能监控 权限模拟

模式切换通过顶部工具栏按钮触发，不影响已编辑内容。所有模式共用同一 PageSchema 存储格式。
