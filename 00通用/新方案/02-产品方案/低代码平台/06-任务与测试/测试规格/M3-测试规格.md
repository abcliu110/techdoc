# M3 测试规格

> 版本：v0.1
> 范围：T-301~T-320
> 目标：验证工作流、插件、导入导出、Connector、通知、商用业务能力、竞品补全 M3 能力、License 和私有化边界。
> ADR 门禁：PRD 不变量、安全与兼容边界始终必测；PERM、OUTBOX、OBJECT-EXT、CONVERSION、FLEXORG、APP-PACKAGE 等 proposed ADR 的精确对象、字段、状态和算法，只有在对应 ADR/等价决策记录 accepted 后才进入阻断性验收。未终审项不得用于声明 M3 完成。

---

## 1. T-301 工作流测试

| 测试 | 必测 |
|---|---|
| `WorkflowInstanceVersionTest` | 在途实例钉住旧版本 |
| `WorkflowApprovalTaskTest` | 审批任务创建、认领、完成 |
| `WorkflowNodeFailureTest` | 节点失败、重试、人工介入 |
| `WorkflowImpactReportTest` | 删除角色/字段/节点生成影响报告 |
| `WorkflowMetricTest` | 流程失败率、平均耗时 |

## 2. T-302/T-303 插件测试

| 测试 | 必测 |
|---|---|
| `PluginManifestSignatureTest` | 签名不合法拒绝 |
| `PluginLifecycleTest` | install/enable/disable/upgrade/rollback/uninstall |

## 7. T-311~T-320 竞品补全 M3 能力测试

| 测试 | 必测 |
|---|---|
| `ConvertRuleSingleStepTest` | 1:1 单级下推生成目标单并记录 LinkTrace |
| `ConvertRuleIdempotencyTest` | 同一源单重复下推按幂等策略处理 |
| `ConvertRulePermissionTest` | 下推前校验源 read/action 与目标 create 权限 |
| `PrintTemplateRenderTest` | 服务端渲染 HTML/PDF，模板版本和 metaHash 入审计 |
| `PrintTemplateInjectionTest` | script/iframe/远程资源/事件属性被阻断 |
| `PrintTemplateMaskingTest` | 打印字段按 AccessView 和 MASKED 裁剪 |
| `WebhookUrlGuardTest` | webhook URL 阻断 localhost、私网、metadata 和 DNS rebinding |
| `WebhookSignatureRetryTest` | 签名、重试、死信和人工重放 |
| `ApiTokenScopeTest` | token scope 只能缩小主体权限，不能绕过 AccessView |
| `ApiTokenLifecycleTest` | 创建只显示一次明文、轮换、撤销、过期和 lastUsedAt |
| `AggregateViewPermissionTest` | group by 先合并 data_scope，聚合不泄露无权行 |
| `GlobalSearchPermissionTest` | 无权限对象/字段不参与检索和摘要 |
| `CommentXssTest` | 评论内容 XSS 净化，危险链接和未授权附件被拒 |
| `TimelineAccessTest` | 用户失去记录权限后不再看到历史评论和时间线 |
| `SsoOrgSyncConflictTest` | 外部组织删除/重命名/合并生成同步计划和冲突报告 |
| `SsoDisabledUserSessionTest` | 禁用用户旧会话在最大窗口内失效 |
| `RecycleBinRestoreTest` | 恢复前校验唯一约束、引用和状态，冲突报告可读 |
| `RecycleBinPurgeAuditTest` | 彻底删除二次确认、审计和保留期策略 |
| `TemplateLibraryInstallTest` | CRM/请假/采购模板复用应用包安全扫描和兼容矩阵 |
| `TemplateUpgradeMergeTest` | 模板升级不覆盖客户扩展，冲突阻断并出报告 |

## 8. M4 边界回归测试

| 测试 | 必测 |
|---|---|
| `MobileReservedBoundaryTest` | M3 不启用完整移动端，但 PageSchema 不绑定固定像素作为唯一布局 |
| `EnvironmentBoundaryTest` | M3 只允许独立部署实例 + 应用包迁移，不出现租户内 dev/test/prod 运行时入口 |
| `CrossAppReferenceBoundaryTest` | M3 阻断运行态跨 app link，仅保留 ADR 和扩展位 |
| `ScriptSandboxBoundaryTest` | 用户自由脚本入口不可见，表达力只走表达式/规则/ConvertRule/插件 SPI |
| `ConnectorLibraryBoundaryTest` | 内置连接器不得绕过 Connector SPI，50+ 连接器生态不作为 M3 验收 |
| `I18nBoundaryTest` | 中文交付可用，元数据多语言 JSON 和 i18n 资源目录存在，但不承诺完整语言包切换 |
