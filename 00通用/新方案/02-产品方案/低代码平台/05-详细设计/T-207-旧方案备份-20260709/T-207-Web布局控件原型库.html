T-207 Web 布局控件原型图

这版展示的是“布局控件”本身，不是页面布局模板。每个控件都按设计器组件表达：工具箱入口、画布中的控件边界、内部插槽/投放区、尺寸手柄、属性协议，并覆盖常用 Web 布局。

控件边界 Slots / DropZone 设计态 Overlay 属性协议
