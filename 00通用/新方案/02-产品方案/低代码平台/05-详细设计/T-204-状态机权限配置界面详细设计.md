# T-204 状态机与权限配置界面详细设计

> 版本：v0.1
> 里程碑：M2
> 适用任务：T-204
> 依据：`../03-需求/PRD-产品需求规格说明书.md` REQ-030~043；`../04-架构决策/02-运行时引擎设计.md`、`../04-架构决策/03-设计器与前端设计.md`
> 5A 状态：baseline-approved。`../04-架构决策/UI-and-Component-Decision-Matrix.md` 已人工接受；本设计作为 M2 UI 详细设计基线，禁止提前在 M0/M1 实现。

## 1. 目标

让设计者通过图形化状态机和权限矩阵配置 M1 已实现的状态、动作、规则和 AccessView 权限。

## 2. 状态机设计器

界面：

```text
状态节点画布
流转连线
右侧属性面板
校验结果面板
模拟执行面板
```

状态属性：

```text
code/name/stateType/docEditable/allowedUpdateFields
```

流转属性：

```text
actionCode/fromState/toState/allowedRoles/conditionExpression/postRuleCodes
```

## 3. 权限矩阵

矩阵维度：

```text
Role x Object x Operation
```

单元格抽屉配置：

```text
dataScope
scopeCondition
fieldPermissions
actionCodes
export/import
attachment
```

字段权限以字段列表方式展示，支持按 permLevel 批量授权。

## 4. 模拟与解释

提供设计态模拟：

```text
选择角色
选择对象
选择操作
可选记录 owner 快照
输出 AccessView 和状态动作可见性
```

模拟结果仅用于设计验证，不作为运行态授权依据。

## 5. 校验

发布前阻断：

1. 无 initial 状态或多个 initial 状态。
2. transition 引用不存在状态、动作、角色。
3. final 状态仍配置普通编辑动作。
4. 权限矩阵缺少运行角色基础 read 权限但页面引用该角色。
5. dataScope custom 表达式非法或引用无权限字段。

## 6. 验收

1. 能配置“草稿 -> 已提交 -> 已审批/已驳回”。
2. 经理角色可审批，销售不可审批。
3. 文员金额字段只读。
4. 模拟执行能解释销售为什么只能看自己的订单。
5. 配置错误在发布前阻断。

### 7.3 UI 设计系统对照

状态机设计器使用 reactflow 画布 + AntD 属性面板/抽屉；权限矩阵使用 AntD Table + Drawer。复杂画布必须提供列表式替代编辑入口，不能只依赖拖拽。

### 7.4 可访问性规则

| 规则 | 验收方式 |
|---|---|
| 状态机画布有键盘替代路径 | 可通过状态/流转列表新增、编辑、删除节点和连线 |
| 权限矩阵可键盘导航 | 单元格进入抽屉、勾选、保存均可键盘完成 |
| 模拟结果可读 | 输出解释使用文本链路，不只靠颜色或连线 |
| 错误阻断可定位 | 发布错误能定位到状态、动作、角色、字段或 dataScope 表达式 |

### 7.5 状态组合

| 权限状态 | 数据状态 | 交互状态 | 配置界面行为 |
|---|---|---|---|
| design none | any | any | 不显示配置入口 |
| design read | published | idle | 只读查看状态机和权限矩阵 |
| design write | draft/dirty/error | editing/submitting | 可编辑草稿；发布前执行 blocking 校验 |
| runtime simulate | stale/conflict | loading/conflict | 模拟只做解释，不写运行态授权 |

### 7.6 失败模式

| 失败模式 | 处置 |
|---|---|
| 无 initial 或多个 initial | 发布前阻断 |
| transition 引用不存在状态、动作、角色 | 发布前阻断并定位引用 |
| 权限矩阵放宽字段权限导致页面越权 | 以 AccessView 发布校验为准，阻断并输出 explain |
| 自定义 dataScope 表达式非法 | 保存 warning，发布 blocking；表达式不得字符串拼接 |
| 模拟结果与运行时不一致 | 模拟只作诊断；以运行时 AccessView 为准并记录 drift 测试失败 |
