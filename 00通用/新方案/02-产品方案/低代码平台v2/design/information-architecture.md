# 工作台信息架构

```text
顶部：产品/文档上下文 + 保存历史 + 预览 + 发布准备
  左活动栏：对象 / 页面 / 规则 / 权限 / 预览
    左资源面板：当前工作区的对象、字段、组件、大纲、规则或角色
      中央：上下文工具栏 + 真实页面画布 + 分析器
    右属性面板：属性 / 布局 / 规则 / 权限
底部：当前节点路径 + Schema/保存/服务状态
```

## 导航规则

- 工作区切换不改变当前业务对象和页面文档。
- 画布、资源树、属性面板和底部路径共享 `selectedNodeId`。
- 规则和权限可以从字段属性直接跳转，但必须保留返回上下文。
- 预览是模态运行态，不替换设计文档。
- 发布准备是检查结果，不等于正式发布。

## 密度规则

- 工具栏和导航采用固定高度，选中态不得改变尺寸。
- 面板标题使用 13–15px，普通 UI 使用 11–13px。
- 长字段名、业务对象名和路径使用单行省略并保留 title/上下文。
- 不使用嵌套卡片、渐变、装饰背景或营销式大标题。
