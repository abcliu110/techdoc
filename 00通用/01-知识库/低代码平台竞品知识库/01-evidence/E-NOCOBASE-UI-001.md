# 证据：NocoBase 页面构建与 Table Block 可见界面

## 原始观察

NocoBase 官方 2.0 教程第 3 章展示“Building Pages”过程：进入 UI Editor 后，通过顶部和侧边菜单新增页面，在空白页面点击 Add block，选择 Data blocks -> Table，并为 Tickets collection 添加 Table block。

页面中包含官方截图证据，展示 Add block、Table block、字段显示配置、Filter Form、Table block 布局拖拽、Categories tree table 等可见界面状态。

## 证据强度

直接事实：官方教程页面明确展示页面构建入口、Table block 添加步骤和截图。

## 可抽取知识

- NocoBase 页面构建不是纯自由画布，而是以 block 为页面构成单元。
- Table block 与 collection 绑定，是从数据模型生成页面视图的关键 UI 模式。
- Filter Form 与 Table block 可组合并拖拽布局，说明其页面编辑器支持块级布局编排。
