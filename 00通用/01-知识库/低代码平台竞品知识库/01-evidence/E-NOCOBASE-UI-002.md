# 证据：NocoBase 表单与详情可见界面

## 原始观察

NocoBase 官方 2.0 教程第 4 章展示 Forms & Details：在 All Tickets 页面中通过 Actions 启用 Add new 操作，点击后打开弹窗；弹窗内通过 Add block -> Data blocks -> Form (Add new) 添加表单，并选择字段。

官方页面包含多张截图，覆盖 Add new 操作按钮、弹窗中的 Form block、字段选择、表单模板保存、Edit Form、行内快速编辑等界面。

## 证据强度

直接事实：官方教程页面以截图和步骤展示表单新增、编辑、模板复用和快速编辑入口。

## 可抽取知识

- NocoBase 将“动作按钮”作为连接列表和表单弹窗的页面机制。
- Form block 可按 collection 字段生成，并允许字段选择、布局调整和模板复用。
- 同一业务对象的新增表单和编辑表单可通过模板保持一致。
