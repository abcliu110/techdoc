# UI 设计：金蝶表单设计器像素级参考

## 使用边界

本卡只作为自研低代码平台的设计参考，不用于照搬金蝶界面。结论来自金蝶官方开发者社区公开文档截图的像素级观察，重点抽取布局比例、视觉密度、交互层级和控件语法。

## 设计器壳层

| 区域 | 观察值 | 设计启发 |
|---|---:|---|
| 顶部工具栏 | 约 50-58px 高，蓝色底，白色线性图标 + 文字 | 高频编辑命令集中在顶部，适合图标按钮，不宜做大按钮 |
| 左控件栏 | 约 265-290px 宽 | 物料区可承载搜索、分类和三列控件网格 |
| 右属性栏 | 约 275-300px 宽 | 属性配置适合固定窄栏，避免弹窗打断编辑流 |
| 中央画布 | 白底，占主要空间，左右有浅灰分隔 | 画布必须保持干净，选中态再显示控制柄 |
| 底部路径 | 约 24-28px 高，面包屑式层级 | 当前选中层级要可见，便于定位嵌套容器 |

## 控件库形态

左侧控件库不是大卡片，而是高密度图标网格：

- 分类标题约 28-32px 高，带展开箭头。
- 控件项约 70-85px 宽，三列排列。
- 图标约 24-32px，采用细线蓝灰风格。
- 文案 12-13px，灰色或深灰，放在图标下方。
- 顶部有搜索框，输入高度约 28-32px。
- 常用分类包括通用字段、业务字段、通用控件、布局/容器等。

对自研平台的启发：控件库应强调扫描效率。字段较多时优先提供搜索、分类折叠、最近使用和对象字段过滤，而不是把物料做成营销式卡片。

## 属性面板形态

右侧属性面板采用 tab + 表单配置：

- 标题栏为蓝色，标题约 12-14px。
- 页签包含“业务 / 样式 / 布局”，活动页签使用蓝色下划线。
- 属性行多为左标签、右输入，标签列约 60-75px。
- 输入框高度约 26-30px，边框为浅灰。
- 下拉、单位、更多操作使用右侧灰蓝色小块按钮。
- 布局页包含缩略模板、开关、下拉、效果预览。
- 样式页包含字体、字重、颜色、边距、圆角、宽高等配置。
- 业务页承载标识、名称、元素类型、可见性、锁定性、控件方案等语义属性。

对自研平台的启发：属性面板必须按“业务语义 / 视觉样式 / 布局规则”分层。业务规则、权限、插件、操作不应塞进样式页。

## 画布编辑态

真实截图中，画布选中反馈非常明确：

- 选中区域使用 1px 蓝色边框。
- 大容器选中时可出现虚线或整块蓝框。
- 控件右上角显示小型快捷按钮，如新增、删除。
- 拖入控件后，画布中的控件保持运行态近似外观。
- 底部面包屑同步显示选中路径，例如表单 -> 控件。

对自研平台的启发：画布不需要复杂装饰，但必须有强选中态、层级定位和快捷操作。嵌套容器越多，越需要大纲树和底部路径辅助。

## 字段控件运行态

金蝶字段控件整体偏企业后台密集风格：

- 默认字段是标签 + 下划线，不是完整边框输入框。
- 边框模式作为另一种字段风格存在。
- 激活态底线蓝色，错误态底线红色。
- 必录用红色星号 + 错误文案，错误文案在字段下方。
- 禁用态使用浅灰底或浅灰字。
- 下拉浮层白底轻阴影，选中项浅蓝底。
- 字段布局面板支持两列排布，容器用浅灰边框或虚线边框提示。

对自研平台的启发：后台录入优先使用高密度字段样式。表单设计器中可以保留运行态预览，但编辑态要叠加选中边框、拖拽边界和操作按钮。

## 视觉语言

| 项 | 观察 |
|---|---|
| 主色 | 蓝色工具栏和选中态，接近企业软件常用蓝 |
| 背景 | 画布白色，侧栏浅灰，分隔区浅灰 |
| 边框 | 多为 1px 浅灰，选中为 1px 蓝色 |
| 圆角 | 很克制，多数输入和按钮接近 0-2px |
| 图标 | 线性图标，小尺寸，功能优先 |
| 字号 | 控件库/属性栏多为 12-14px，正文表单字段约 14px |
| 密度 | 高密度、工具型、面向长期操作，不是营销页面风格 |

## 自研平台建议

1. 采用五区结构：顶部命令栏、左物料/大纲、中画布、右属性、底部层级路径。
2. 左侧物料区提供“对象字段”和“通用控件”双来源，字段拖入后自动绑定数据模型。
3. 右侧属性按业务、规则、样式、布局、权限拆分；首版至少保留业务 / 样式 / 布局。
4. 控件运行态和设计态应同构：设计态叠加蓝色选择框、控制柄和大纲同步。
5. 字段默认使用紧凑下划线样式，支持边框、禁用、必录、错误、Tips、按钮跟随等状态。
6. 顶部工具栏以图标命令为主，提供保存、预览、撤销重做、复制粘贴、对齐、辅助线等能力。
7. 不照搬金蝶的具体图标、颜色值、文案和布局模板图；吸收其企业级密度、元数据分层和编辑器信息架构。
