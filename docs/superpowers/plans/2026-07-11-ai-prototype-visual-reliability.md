> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Strengthen the unified prototype engineering method so AI-generated prototypes prevent, detect, classify, repair, and re-verify broad visual reliability defects rather than treating overflow as an isolated symptom.

**Architecture:** Keep the existing document as the single normative authority. Add a non-skippable visual reliability floor, a seven-layer validation loop, stronger visual contracts and geometry requirements, root-cause repair routing, detector certification fixtures, and objective promotion criteria from candidate method to verified execution baseline.

**Tech Stack:** Markdown normative specification, browser geometry concepts, Playwright-oriented verification contracts.
