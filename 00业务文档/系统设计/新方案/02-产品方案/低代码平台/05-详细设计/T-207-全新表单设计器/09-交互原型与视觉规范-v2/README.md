# T-207 表单设计器原型 V2

## 版本边界

- `../09-交互原型与视觉规范`：V1 冻结版本，不再修改。
- 当前目录：V2 重新设计版本。
- V2 保留 V1 的 PT / IX / PS 编号、业务状态和自动化抓手，仅重构可见界面与真实交互。

## 当前验收顺序

1. PT-001 工作台整体布局。
2. 用户完成 PT-001 人工检查后，再依次重构后续 PT。
3. 每张原型必须通过 1440×900、1920×1080 运行态视觉门禁和关键交互验证。

## V2 设计原则

- 面向企业 PC Web WYSIWYG 表单 / 单据设计器，不做报表、问卷或营销页面。
- 主框架参考 Visual Studio / DevExpress 的工具工作台层级，吸收金蝶的业务对象与单据设计语义。
- 固定尺寸工具控件不得因选中、悬停、文字或动态状态发生布局跳变。
- 每个可见入口必须有真实面板、稳定抓手和可观察状态。
- 任何裁切、重叠、逐字换行、离屏操作或伪交互均为阻断问题。
