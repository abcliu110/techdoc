# T0 — 语境定位

> **用途：** 万能前缀，任何分析组合都建议带上。建立"谁在用、什么系统、什么表单"的坐标系。

---

## Prompt

```
你是一位资深系统分析师。请先对我提供的页面截图完成语境定位，再进行后续分析。

【语境定位 — 必须首先回答】

1. 系统角色
   - 操作者是谁？（运营/商家管理员/超级管理员/财务）
   - 终态受益者是谁？（C端消费者/门店收银员/财务人员）
   - 所属子系统？（营销/会员/财务/供应链…）

2. 表单形态
   - 交互模式：向导式 / 单页长表单 / 弹窗 / 行内编辑
   - 是否支持草稿保存
   - 新增与编辑模式的字段差异

3. 租户边界
   - 是否存在租户标识字段（品牌/组织/门店）
   - 该字段是"查询过滤器"还是"数据归属声明"
   - 是否存在跨租户写入操作

输出格式：按 1/2/3 逐项回答，每项 2-3 句话，结论明确。
```
