## 深入探索记录：「加入礼品」弹窗（来自清单第13项）

弹窗标题：「加入礼品」

### 字段清单

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 礼品名称 | 单选下拉(带搜索) | ● | 按分组显示优惠券（如测试分组：折扣券测试、40元、30元、20元、10元、门店测试券、阿翔正餐测试券） |
| 礼品编码 | 文本(只读) | - | 选择礼品后自动填充 |
| 每人每次发放张数 | 数字输入 + 张 | ● | - |
| 有效期类型 | Radio 4选项 | - | 见下方详情 |
| 有效天数 | 数字输入 + 天 | ● | 仅「相对有效期」时显示 |
| 有效期 | 日期范围选择器 | ● | 仅「固定有效期」时显示 |
| 生效类型 | Radio 3选项 | - | 按天/按小时/按天零点，三者界面无差异 |
| 何时生效 | 单选下拉 | - | 5个选项，见下方详情 |
| 是否计入实收 | Radio 2选项 | - | 见下方详情 |

### 「有效期类型」Radio 联动

| 选项 | 界面变化 |
|------|---------|
| 相对有效期 | 显示「有效天数」数字输入框(必填) |
| 固定有效期 | 显示「有效期」日期范围选择器(开始日期→结束日期，必填) |
| 自然月内有效 | 无额外字段 |
| 永久有效期 | 无额外字段 |

### 「何时生效」下拉选项

| 选项 | 界面变化 |
|------|---------|
| 立即生效 | 无 |
| 1天后生效 | 无 |
| 2天后生效 | 无 |
| 3天后生效 | 无 |
| 自定义 | 出现数字输入框 + 「天后生效」 |

### 「是否计入实收」Radio 联动

| 选项 | 界面变化 |
|------|---------|
| 不计入 | 无额外字段 |
| 计入 | 出现「实收金额」下拉框 |

### 「实收金额」下拉选项

| 选项 | 界面变化 |
|------|---------|
| 优惠券核销金额 | 无额外输入 |
| 核销金额的 | 出现百分比输入框 + 「%」 |
| 自定义金额 | 出现金额输入框 + 「元」 |
