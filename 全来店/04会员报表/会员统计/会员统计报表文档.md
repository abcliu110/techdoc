# 会员统计报表

## 1. 报表概览

**会员统计报表**位于全来店商户中心的`会员中心 → 会员报表 → 会员统计`路径下，用于统计和分析会员相关的各项数据。

## 2. 功能区域

### 2.1 筛选条件
- **会员方案**：下拉选择框，可选择不同的会员方案
- **入会日期**：下拉选择框，可按日期筛选
- **入会门店**：下拉选择框，可按门店筛选
- **查询**：绿色按钮，执行查询操作
- **高级筛选**：灰色按钮，打开高级筛选选项
- **重置**：灰色按钮，重置所有筛选条件
- **导出**：右上角按钮，导出报表数据

### 2.2 统计维度
报表支持以下四个核心维度进行数据统计：

1. **会员方案**：按会员卡类型进行统计
2. **会员等级**：按会员等级（如普通会员、银卡会员、金卡会员）进行统计
3. **会员**：按单个会员实体进行统计
4. **会员卡**：按具体的会员卡进行统计

## 4. 报表特点

1. **多维度统计**：涵盖储值金额、次数、卡数量、会员数量等多个维度
2. **财务明细**：详细展示本金、赠金、可用余额等财务数据
3. **状态监控**：包含冻结金额等状态信息
4. **汇总功能**：底部自动计算合计数据
5. **筛选功能**：支持按会员方案、日期、门店进行筛选
6. **导出功能**：支持报表数据导出
7. **维度切换**：支持在会员方案、会员等级、会员、会员卡四个维度间切换

## 6. 报表用途

1. **会员管理**：了解会员规模和分布情况
2. **财务分析**：掌握储值资金流动和余额情况
3. **营销决策**：基于数据制定会员营销策略
4. **运营监控**：实时监控会员体系运行状态
5. **数据导出**：支持进一步的数据分析和报表制作
6. **多维度分析**：通过切换不同维度，从不同角度分析会员数据

## 7. 操作指南

### 7.1 切换统计维度
1. 点击左侧的下拉菜单
2. 选择需要的统计维度（会员方案/会员等级/会员/会员卡）
3. 系统会自动刷新数据

### 7.2 筛选数据
1. 选择会员方案、入会日期、入会门店等筛选条件
2. 点击【查询】按钮执行筛选
3. 如需更详细的筛选条件，点击【高级筛选】

### 7.3 导出报表
1. 点击右上角的【导出】按钮
2. 选择导出格式（如Excel、CSV等）
3. 保存导出的文件

## 8. 注意事项

1. 报表数据为实时统计，反映当前系统中的会员数据
2. 导出的报表文件可用于离线分析和存档
3. 高级筛选功能可根据需要设置更复杂的筛选条件
4. 切换维度后，数据会重新计算，可能需要一定时间加载
