# 会员储值 - 每个字段详细说明

## crm_deposit_plan（储值方案主表）

### 系统字段
| 字段 | 类型 | 长度 | 必填 | 约束 | 说明 |
|------|------|------|------|------|------|
| pid | BIGINT | 20 | ✓ | PRIMARY KEY, AUTO_INCREMENT | 物理编号，数据库自增主键 |
| mid | BIGINT | 20 | ✓ | NOT NULL, INDEX | 商户ID，标识数据所属的商户 |
| sid | BIGINT | 20 | ✓ | NOT NULL, INDEX | 门店ID，标识数据所属的门店 |
| lid | BIGINT | 20 | ✓ | NOT NULL, UNIQUE INDEX | 逻辑编号，使用雪花算法生成，业务上的唯一标识 |

### 方案基础信息
| 字段 | 类型 | 长度 | 必填 | 说明 | 示例值 |
|------|------|------|------|------|-------|
| name | VARCHAR | 100 | ✓ | 储值方案名称，用于展示和识别 | "新人首充礼", "周末超值储值" |
| description | TEXT | - | ✗ | 方案说明，营销文案，可包含HTML格式 | "新会员首次储值送50%赠送金额" |
| image_url | VARCHAR | 255 | ✗ | 方案展示图片URL，用于前端展示 | "https://cdn.xxx.com/deposit-plan.jpg" |
| sort_no | INT | 11 | ✗ | 排序号，用于展示的排序顺序 | 1, 2, 3（越小越靠前） |

### 方案分类
| 字段 | 类型 | 长度 | 必填 | 说明 | 说明 |
|------|------|------|------|------|------|
| plan_type | TINYINT | 4 | ✓ | 方案类型 | 1=储值分期送礼, 2=智能储值, 3=储值套餐 |
| category_lid | BIGINT | 20 | ✗ | 分类ID，关联 crm_deposit_category.lid | 100001（对应某个分类） |

### 状态管理
| 字段 | 类型 | 长度 | 必填 | 说明 | 说明 |
|------|------|------|------|------|------|
| status | TINYINT | 4 | ✓ | 方案状态 | 0=禁用（不展示、不可购买）, 1=启用（正常展示） |

### 有效期控制
| 字段 | 类型 | 长度 | 必填 | 说明 | 示例值 |
|------|------|------|------|------|--------|
| date_type | TINYINT | 4 | ✓ | 日期类型 | 1=指定日期范围, 2=永久有效 |
| begin_time | DATETIME | - | ✗ | 开始时间，当 date_type=1 时必填 | 2026-03-20 09:00:00 |
| end_time | DATETIME | - | ✗ | 结束时间，当 date_type=1 时必填 | 2026-03-31 23:59:59 |
| | | | | 注意：end_time 必须晚于 begin_time | |

### 协议相关
| 字段 | 类型 | 长度 | 必填 | 说明 | 示例值 |
|------|------|------|------|------|--------|
| agreement_enabled | TINYINT | 4 | ✓ | 是否启用协议 | 0=不启用, 1=启用 |
| agreement_lids | JSON | - | ✗ | 协议ID列表（JSON数组） | [100001, 100002] |
| | | | | 当 agreement_enabled=1 时可选指定协议 | |
| | | | | 关联 crm_deposit_agreement.lid | |

### JSON规则字段 - 时间相关

#### cycle_rule（周期规则）
| 结构 | 说明 | 示例 |
|------|------|------|
| type | 周期类型 | DAILY/WEEKLY/MONTHLY |
| values | 对应的值列表 | WEEKLY: [1,2,3,4,5] 表示周一到周五 |
| | | MONTHLY: [1,15] 表示每月1号和15号 |

**完整例子：**
```json
{
  "type": "WEEKLY",
  "values": [1, 2, 3, 4, 5]  // 仅工作日（周一到周五）
}
```

#### time_periods（时段规则）
| 字段 | 说明 | 示例 |
|------|------|------|
| type | "ALL_DAY" 或 "SPECIFIED" | "SPECIFIED" 表示指定时段 |
| periods[] | 时段数组 | |
| startMin | 开始时间（分钟数） | 540 表示 9:00（9*60） |
| endMin | 结束时间（分钟数） | 1020 表示 17:00（17*60） |

**完整例子：**
```json
{
  "type": "SPECIFIED",
  "periods": [
    {
      "startMin": 540,   // 09:00
      "endMin": 720      // 12:00
    },
    {
      "startMin": 900,   // 15:00
      "endMin": 1080     // 18:00
    }
  ]
}
```

#### exclude_dates（排除日期）
| 说明 | 示例 |
|------|------|
| 在这些日期该方案不可用 | ["2026-01-01", "2026-05-01", "2026-10-01"] |
| 日期格式必须是 YYYY-MM-DD | |

### JSON规则字段 - 适用范围控制

#### channels（渠道规则）
```json
{
  "type": "ALL"  // 所有渠道都支持
}

// 或

{
  "type": "SPECIFIED",
  "channels": [
    {
      "code": "POS",
      "name": "门店POS"
    },
    {
      "code": "MINI_PROGRAM",
      "name": "小程序"
    },
    {
      "code": "H5",
      "name": "H5页面"
    }
  ]
}
```

**支持的渠道代码：**
- POS - 门店收银系统
- MINI_PROGRAM - 微信小程序
- H5 - 移动端网页
- APP - 移动应用
- WEB - 电脑网页

#### applicable_orgs（门店/地区规则）
```json
{
  "type": "ALL"  // 所有门店/地区
}

// 或指定门店
{
  "type": "SPECIFIED_STORES",
  "storeIds": [1001, 1002, 1003]
}

// 或指定地区
{
  "type": "SPECIFIED_REGIONS",
  "regionIds": [100, 101]
}
```

#### brand_rule（品牌规则）
```json
{
  "type": "ALL"  // 所有品牌
}

// 或指定品牌
{
  "type": "SPECIFIED",
  "brandIds": [2001, 2002]
}
```

#### member_level_rule（会员等级规则）
```json
{
  "type": "UNLIMITED"  // 不限制会员等级
}

// 或限制特定会员方案和等级
{
  "type": "SPECIFIC_LEVELS",
  "memberPlanLids": [100001, 100002],
  "memberLevels": [
    {
      "memberPlanLid": 100001,
      "memberLevelLids": [3101, 3102]  // 金卡方案中的金牌和白银会员
    },
    {
      "memberPlanLid": 100002,
      "memberLevelLids": [3201]  // 钻卡方案中的钻石会员
    }
  ]
}
```

### JSON规则字段 - 购买限制（最复杂）

#### purchase_limit_rule（购买限制规则）

**完整结构：**
```json
{
  "perUserTotalLimit": {
    "enabled": true,
    "limit": 10,
    "limitMessage": "您已达到购买上限，无法继续购买"
  },
  "perUserPeriodLimit": {
    "enabled": true,
    "periodType": "DAILY",  // DAILY, WEEKLY, MONTHLY, YEARLY
    "limit": 5,
    "limitMessage": "您今日的购买次数已达上限，请明日再来"
  },
  "totalSaleLimit": {
    "enabled": true,
    "limit": 1000,
    "limitMessage": "该方案已售罄，感谢您的支持"
  },
  "totalSalePeriodLimit": {
    "enabled": true,
    "periodType": "DAILY",
    "limit": 100,
    "limitMessage": "该方案今日已被抢光，请明日再来"
  }
}
```

**字段说明：**
| 限制类型 | 说明 | 场景 |
|---------|------|------|
| perUserTotalLimit | 单个会员的总购买次数限制 | 新人礼只能购买1次 |
| perUserPeriodLimit | 单个会员在周期内的购买次数限制 | 每天最多购买5次 |
| totalSaleLimit | 全局的总销售次数限制 | 方案总共只能销售1000次 |
| totalSalePeriodLimit | 全局在周期内的销售次数限制 | 每天全局最多销售100次 |

### JSON规则字段 - 储值档位规则（最核心）

#### tier_rule（储值档位规则）

**顶级结构：**
```json
{
  "tiers": [
    { /* 档位1配置 */ },
    { /* 档位2配置 */ },
    { /* 档位3配置 */ }
  ]
}
```

**单个档位（TierConfig）详细说明：**

| 字段 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| saveAmount | BigDecimal | ✓ | 储值本金，必须 > 0 | 100.00 |
| giftAmount | BigDecimal | ✗ | 赠送金额，必须 >= 0 | 50.00 |
| giftPoints | BigDecimal | ✗ | 赠送积分，必须 >= 0 | 100 |
| recommendTitle | String | ✗ | 推荐标签显示文本 | "热销", "推荐", "限时" |
| isFirstChargeOnly | Boolean | ✗ | 仅首充可用 | true/false |
| isUpgradeMemberLevel | Boolean | ✗ | 是否升级会员等级 | false 表示不升级 |
| upgradeMemberLevels | Array | ✗ | 升级配置列表，当上字段=true时必填 | 见下表 |
| isLevelDelay | Boolean | ✗ | 是否延期等级有效期 | true/false |
| levelDelayDays | Integer | ✗ | 延期天数，当 isLevelDelay=true 时必填 | 30 |
| isGiftEnabled | Boolean | ✗ | 是否启用赠送礼品 | 一级开关 |
| coupons | Array | ✗ | 赠送优惠券列表，当 isGiftEnabled=true 时可选 | 见下表 |
| couponPackages | Array | ✗ | 赠送券包列表，当 isGiftEnabled=true 时可选 | 见下表 |
| issueRuleType | Enum | ✗ | 发放规则类型 | ONE_TIME/PERIODIC |
| issueCycleType | Enum | ✗ | 周期类型，当 issueRuleType=PERIODIC 必填 | DAILY/WEEKLY/MONTHLY |
| issueCycleValues | Array | ✗ | 周期值列表，当 issueRuleType=PERIODIC 必填 | [1,2,3] 或 [1,15,20] |
| issueTime | LocalTime | ✗ | 发放时间，当 issueRuleType=PERIODIC 必填 | "20:00" |
| issueTimes | Integer | ✗ | 总发放次数，当 issueRuleType=PERIODIC 必填 | 12 |
| firstIssueImmediate | Boolean | ✗ | 首次是否立即发放，当 issueRuleType=PERIODIC 必填 | true |
| sortNo | Integer | ✗ | 档位排序号 | 1, 2, 3 |

**升级会员等级配置（UpgradeMemberLevelConfig）：**

**赠送优惠券项（CouponItem）：**

| 字段 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|

| issueQtyPerTime | Integer | ✓ | 每次发放数量 | 1, 2, 3 |

| termOfValidityMethod | Enum | ✓ | 有效期类型 | XD=相对, GD=固定, ZRY=自然月, YJ=永久 |
